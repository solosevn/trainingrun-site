TRAgents Methodology

Measuring how well AI models perform as autonomous agents

Version 1.0 | February 2026

Section 1

What is TRAgents?

TRAgents is the interactive daily agentic AI benchmark tracker. It tracks composite TRAgents scores measuring how well AI models perform as autonomous agents across real-world tasks, updated daily.

Unlike TRSbench (which measures overall capability) or TRUscore (which measures truthfulness), TRAgents measures whether a model can be trusted to act autonomously — reliably, safely, and affordably.

Section 2

The TRAgents Formula

Composite Score
TRAgents = Task Completion (25%) + Cost Efficiency (20%) + Tool Reliability (20%) + Safety & Security (15%) + Accessibility (10%) + Multi-Model Efficiency (10%)
Pillar Weight What It Measures Primary Sources
Task Completion 25% Can the agent finish multi-step real-world jobs correctly? GAIA, SWE-bench Verified, OSWorld, tau-squared-bench
Cost Efficiency 20% Cost per successfully completed task ($/task) ARC-AGI-2, Artificial Analysis
Tool Reliability 20% Correct tool calls, API usage, no hallucinated actions MCP Atlas (Scale AI), Galileo Agent Leaderboard
Safety & Security 15% Prompt injection resistance, safe autonomous behavior MASK Benchmark (Scale AI)
Accessibility 10% Open-weight availability, hardware reqs, framework support Ollama Library
Multi-Model Efficiency 10% Performance in router/worker configurations OpenRouter Rankings
Section 3

Why These Weights?

Task Completion 25%

The fundamental question. If the agent can't finish the job, nothing else matters.

Cost Efficiency 20%

The #1 practical concern. Agents consume 5–20x more tokens than simple chat. Multi-model routing cuts costs 60–80%.

Tool Reliability 20%

What separates agents from chatbots. A model can be intelligent enough to plan the right steps but fail when it tries to call tools.

Safety & Security 15%

Agents take actions — and when they fail, they fail catastrophically. 900 malicious skills found on ClawHub, a one-click RCE exploit affected 135,000+ instances.

Accessibility 10%

Answers “Can I actually run this?” Binary qualifier more than performance gradient.

Multi-Model Efficiency 10%

How the real world deploys agents. NVIDIA's Orchestrator-8B proved that a trained router beats GPT-5 at 30% of the cost.

Section 5

Qualification Rules

A model must have non-null scores in at least 3 of the 6 TRAgents pillars to appear on the leaderboard.

Models with null pillars are scored on available data only, with weights normalized proportionally.

Discovery-first: every daily update scans ALL models on ALL agentic benchmark source platforms.

Section 6

Data Integrity

SHA-256 checksum verification is built into the scoreboard page.

Checksum is computed from the canonical string of model names and scores.

If the integrity check fails, a warning banner appears on the scoreboard.
