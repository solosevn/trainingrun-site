The Specialists Sources

Category 1: CODING

Identifies the AI model that solves the most coding problems correctly using contamination-free benchmarks.

Metric: LiveCodeBench Score (% of problems solved correctly)

Primary Data Source

LiveCodeBench via Artificial Analysis
LiveCodeBench tests models on coding problems released AFTER model training cutoffs, eliminating data contamination concerns that plague other benchmarks.

→ artificialanalysis.ai/evaluations/livecodebench

What It Measures

Percentage of coding problems solved correctly on first attempt. Higher = better coder.

How to Verify

  1. Visit artificialanalysis.ai/evaluations/livecodebench
  2. Find the model with the highest score (pass@1 %)
  3. Compare to our featured Coding Specialist

Category 2: HUMAN PREFERENCE

Identifies the AI model most preferred by real humans in blind, side-by-side comparisons (helpfulness, coherence, safety, etc.).

Metric: Chatbot Arena Elo Rating (higher = more preferred)

Primary Data Source

LMSYS Chatbot Arena (now LMArena/Arena.ai)
Crowdsourced Elo from millions of anonymous human votes on pairwise blind battles—no company influence, pure preference.

→ lmarena.ai/?leaderboard (or chat.lmsys.org/?leaderboard for legacy view)

What It Measures

Elo score from real user votes (typically 1000–1500+ range for top models). Reflects practical "which one would you use?" preference.

How to Verify

  1. Visit lmarena.ai/?leaderboard
  2. Sort by Elo rating and find the top model
  3. Compare to our featured Human Preference Specialist

Category 3: BEST VALUE

Identifies the most cost-effective model (performance per dollar spent on inference).

Metric: Value Score (Intelligence/Efficiency ÷ Price per M tokens; tiers: Best / Great / Good)

Primary Data Source

Artificial Analysis Model Pricing & Efficiency
Independent pricing (USD/M tokens), output speed (tokens/s), and intelligence index. Blended for value (e.g., TRS × tps / price).

→ artificialanalysis.ai/models (filter by Price, Speed, Intelligence)

What It Measures

Cost-efficiency: High performance (TRS/intelligence) at low inference cost. Tiers based on calculated score thresholds.

How to Verify

  1. Visit artificialanalysis.ai/models
  2. Sort/filter by Price, Output Speed, Intelligence Index
  3. Calculate value (e.g., Intelligence × Speed ÷ Price) and compare tiers to our ranking
