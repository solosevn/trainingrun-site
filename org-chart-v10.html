Notes:
• TRSitekeeper is the biggest variable — cost scales with how many Telegram interactions you have per day. Heavy usage days could spike it.
• Anthropic API pricing: Claude Sonnet 4.6 = $3/1M input tokens, $15/1M output tokens. Output is the expensive side.
• xAI API pricing: Grok-3 = $3/1M in, $15/1M out. Grok-3-mini = $0.30/1M in, $0.50/1M out. Mini is ~10x cheaper.
• Ollama (Llama 3.1:8b) is completely free — runs on your Mac's GPU/CPU. Cost is only electricity.
• DDP scrapers have zero AI cost. They're pure Python + Playwright browser automation.
• TS Arena agents (Battle Generator, etc.) are not yet built and have no current cost. When built, Battle Generator will use OpenRouter API (variable cost per model).
